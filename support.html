CellFocus Support

CellFocus is an Excel add-in for focused editing of selected cell ranges in a task pane.

Last updated: May 13, 2026

Getting help

For support, bug reports, and feature requests, use the public GitHub issue tracker:

https://github.com/younnieCutler/excel-new-tab-cell-range/issues

Common checks

  • Use Microsoft Excel and open your workbook normally.
  • Select a range, then use the CellFocus ribbon button or context menu command.
  • If the task pane does not appear, confirm that Office add-ins are enabled in your Excel client.
  • If the task pane appears stale, close and reopen the CellFocus pane.

Supported environment

CellFocus is designed for Microsoft Excel with Office Add-ins enabled. The primary workflow is opening an Excel workbook, selecting a range, and editing that range inside the CellFocus task pane.

Service status

CellFocus is hosted as static files on GitHub Pages. If the add-in cannot load, also check GitHub Pages availability and your organization's network policy for Office Add-ins.
